## Session Types

### Home Strength — 5 rounds

Each round:

```
Bicep curls (8kg × 20)   → [Done — reps] or [Skip]
Rest                      → timer up, tap [Ready]
──────────────────────────────────────────────────
Shoulder drill (3kg × 20) → [Done — reps] or [Skip]
Rest                      → same
──────────────────────────────────────────────────
Sit-ups (× 30)            → [Done — reps] or [Skip]
Rest                      → same
──────────────────────────────────────────────────
Push-ups (× 10)           → [Done — reps] or [Skip]
Rest                      → same
──────────────────────────────────────────────────
Plank static (1 min)      → [Done — time] or [Skip]
Rest 5 min (end of round) → same
══════════════════════════════════════════════════
Round 2 of 5 →
```

### Outdoor Strength — 5 rounds

Same structure, pull-ups replace the weights slot:

```
Pull-ups (× 5)       → [Done — reps] or [Skip]
Rest                 → timer up, tap [Ready]
─────────────────────────────────────────────
Sit-ups (× 30)       → [Done — reps] or [Skip]
Rest                 → same
─────────────────────────────────────────────
Push-ups (× 10)      → [Done — reps] or [Skip]
Rest                 → same
─────────────────────────────────────────────
Plank static (1 min) → [Done — time] or [Skip]
Rest 5 min           → same
═════════════════════════════════════════════
Round 2 of 5 →
```

- **Rest timer is a guide, not a judge.** It shows elapsed time counting up. Tap Ready at 0:55 or 1:30 — both are fine. The actual time is logged.
- **Per exercise, not per round.** Hands hurt → skip pull-ups, still do abs. That round records: pull-ups skipped, abs ✓. Nothing collapses into just "round complete."
- **Skip ≠ fail.** Everything is data.
- Session ends when you finish round 5 or tap Stop. Stopped at round 3 → recorded as 3/5.

## Auto-Progression

The app tracks expected training days (Sun/Tue for strength, Wed for run — rest days excluded).

**Rule:** If you hit the current target on every session you showed up to across 2 training-week cycles → target bumps up automatically. Next time you open the session on the Watch, it shows the new number.

- "Showed up" = opened the session and completed at least some exercises
- Full skip (didn't train at all) = doesn't count toward the streak
- Partial session (stopped at round 3, some exercises skipped) = counts as long as you showed up

**Example — pull-ups:**
- Week 1–2: target 5. Hit 5 every Sun + Tue → bumps to 8
- Week 3–4: target 8. Hit 8 every Sun + Tue → bumps to 10
- And so on toward 30

Same logic applies to: push-ups, sit-ups, plank duration, run distance, run pace.

## Progress Calendar

Separate section in the app. Shows every training day:

- Trained ✓ / Skipped ✗ / Partial ~
- Tap any day → see full session detail: each exercise, reps logged, actual rest times, rounds completed
- Per muscle group trend: pull (arms), core (abs/plank), push (chest), run — each tracked independently over time
- Connected to other fitness goals in the app (swimming when it becomes active, etc.)

This is where you see the honest picture — not just "did I train" but "what did my hands do vs my core" across weeks.

## Progression Checkpoints

These are approximate — the app auto-progresses per exercise, but these mark phase shifts:

| Move to next phase when... |
|---------------------------|
| Pull-ups: 8 clean reps every session for 2 weeks |
| Push-ups: 20 clean reps every session for 2 weeks |
